## Nasıl Kullanılır?

Her ders şu yapıya sahiptir:

```
DersXX_Konu/
├── ders_notu.md       → Teorik açıklamalar, kavramlar, diyagramlar
└── ornekler/          → Çalışan Dart/Flutter kod örnekleri
```

### Önerilen Çalışma Yöntemi
1. Önce `ders_notu.md` dosyasını baştan sona oku
2. `ornekler/` klasöründeki kodları incele
3. Örnekleri kendi bilgisayarında çalıştır
4. Alıştırmaları (ders notu sonundaki) kendin yap
5. Bir sonraki derse geç

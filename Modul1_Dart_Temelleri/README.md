## Kod Örneklerini Çalıştırma

Dart'ın bilgisayarınızda kurulu olması gerekir (Flutter ile birlikte gelir):

```bash
# Herhangi bir .dart dosyasını çalıştırmak için:
dart run dosya_adi.dart

# Örnek:
cd Modul1_Dart_Temelleri/Ders01_Degiskenler_ve_Tipler/ornekler
dart run 01_temel_degiskenler.dart
```
